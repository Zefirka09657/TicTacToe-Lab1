## Описание проекта
Проект представляет собой реализацию игры "Крестики-нолики" в двух версиях:

1. **Версия 1.0:** Базовая игра с поочередным вводом координат, проверкой победы и ничьей.
2. **Версия 2.0:** Расширенная версия с подсчетом счета побед и возможностью кастомизации символов.
